阴影之军
越过边境后，我的第一站是利沃夫，这是乌克兰最西部的一座充满石像鬼和哥特式尖顶的古城。它在战前就以其咖啡烘焙厂和啤酒厂而闻名，此后又成为各路志愿者的中转站和信息交流中心。当我在3月22日到达时，这座城市的许多雕像和纪念碑都被用沙袋和塑料袋包裹起来，大教堂的彩色玻璃窗也被封上了。一辆苏联时代的手推车滑过一个中世纪的堡垒，士兵们站在那里抽烟。到了晚上，湿漉漉的鹅卵石在酒吧的霓虹灯下闪闪发光，由于战时禁酒，这些酒吧只限于销售近似啤酒。
一小群讲法语或英语的年轻和中年男子在酒店门口徘徊，他们身着不同程度的战术服装，用手机研究地图。他们很容易成为城里成群结队的记者的猎物，他们忙着提交标题为 "其他人的队伍"、"被诅咒的军团 "和 "卡布奇诺和卡拉什尼科夫 "的报告；或诸如 "我以为我会死"、"我只想杀俄罗斯人 "和 "我不想成为炮灰 "的证词。
与我交谈过的志愿者大多让我感到无知或妄想，与武装力量没有真正的联系。乌克兰人正在运行一个精简的媒体业务，而新闻官员除了在每日简报中向记者发布的内容外，几乎没有其他内容。没有人可以访问前线。指挥官不接受采访。野战医院是禁区。士兵们多次抢走我的手机，并删除了照片和视频。难怪美国地区的报纸，如《特拉华县每日时报》，只能依靠家乡英雄的第一人称，如帕特里克-克里德，一个54岁的哈弗镇人，他告诉该报，他在基辅 "不断清除房屋和建筑中的俄罗斯嫌疑人"。值得称赞的是，《纽约时报》小心翼翼地指出，它未能 "确定任何在乌克兰积极作战的退伍军人"。但其他媒体就没有这么谨慎了。例如，《华盛顿邮报》急于报道一位名叫桑德拉-安德森-艾拉的挪威妇女 "在冲突的大部分时间里一直在前线"。
在《纽约书评》和《纽约》杂志上发表的其他报道，将一些志愿者描绘成反政府的极端分子、嗜血的精神病患者或无望的不满者，但这些报道完全依赖于美国、波兰或离前线数百英里的利沃夫的未来战士的可疑主张和社交媒体的滑稽行为。我意识到，如果有什么东西可以被恰当地称为外国军团，我不会在这里找到它。我需要去基辅。
当时，首都基本上被俄罗斯的坦克和大炮包围，郊区每天都遭到炮击。但不知为何，火车仍在运行。3月27日，我来到利沃夫车站，那里挤满了紧张而疲惫的男人和女人，他们背着行李箱、孩子和宠物。头顶上是一个高高的、华丽的天花板，上面有剥落的石膏拱门和一个我看不懂的西里尔语时刻表。我用手机上的翻译软件买了一张票，并从几个来自加拿大、瑞典和爱尔兰的人那里得到了指示。其中两个人是来打架的，包括一个长相野蛮的瑞典人，他的太阳穴上有一个米老鼠纹身。另外三个是无人机销售员。
他们把我引到空旷的东行月台，那里有士兵在检查证件。在开往基辅的夜车到达时，防空警报响了，但我们还是按计划上了车。我们向东走的人很少。我的车上有一半的铺位都没有人坐。很快，我就快睡着了。
第二天早上，我一走出航站楼，北面就传来了炮声。庄严而华丽的市中心，镶有镀金的城楼和圆顶大教堂，仍然没有受到破坏，但街道上几乎没有人，只有把守检查站和机枪窝的戴面具的士兵。俄国人的攻势有所减弱，但每个人似乎都认为这是一个诡计，围困将重新开始。戒严令已经生效。
在一家尽管有临时禁令但仍在出售品脱啤酒的酒吧，我见到了一位要求匿名的乌克兰高级国防官员。他在拉链夹克下穿了一把斯蒂奇金自动手枪，并给我看了他手机上据称是俄罗斯暴行的照片：一个被压在瓦砾下的死婴；一个在残骸中徘徊的困惑的平民；以及被烧焦的乌克兰士兵的尸体，他声称这是一次白磷攻击的受害者。
我问他国际军团的情况，他说政府没有准备好接收这么多志愿者。其中大约三分之一的人因为缺乏战斗经验而被拒之门外。"他说："他们被杀，而我们却有这样的名声，这不是一件好事。另外三分之一 "在看到真正的战争之后 "离开了，他指的是对亚沃洛夫的打击。他承认，其余的人还没有被组织成一个独立的单位。志愿者们被安置在利沃夫和基辅周围的不同地点，很少有人拥有武器、防弹衣或头盔。他说，在前线有几个经验丰富的老兵，但仅此而已。
他的叙述与我从马修-范迪克（Matthew VanDyke）那里听到的一致，他是我在机场看到的自由职业军事培训师。"国际军团并不存在，"他给我发短信。"这都是宣传，以吸引国际支持、媒体报道，并加强世界对俄罗斯的想法。"在基辅，他在佩尔莫希广场的一家酒店会见了一群大约16名军团成员。他告诉我，"这是你见过的最可悲的一群人，""一辆由失足者组成的小丑车"。"整件事是一个拙劣的伎俩，目的是在媒体上将冲突国际化，"他补充说。"他们希望人们通过大使馆申请，因为他们并不打算真的把他们带到这里。那些自己来的人，他们不知道如何处理。这是一个混乱的局面。"
在圣沃洛迪米尔大教堂（Saint Volodymyr's Cathedral）--据说是矿工、装甲师和炮兵的守护神圣巴巴拉的坟墓--我见到了格鲁吉亚国民军团的指挥官马穆卡-马穆拉什维利，这是一支由外国战士组成的民兵组织--主要来自高加索地区--自2014年以来一直在乌克兰境内。马穆拉什维利是一位魁梧、多毛的东正教基督徒，曾在四场不同的战争中与俄罗斯作战（在格鲁吉亚、车臣、南奥塞梯和乌克兰），有时会来这里祈祷。
我曾听说格鲁吉亚部队容忍各种类型的外国战士--包括来自美国的白人至上主义者--但马穆拉什维利很快予以否认。"我们不接受那些激进的组织，"他说。"我们试图永远不接受他们，新纳粹主义，或种族主义者，或其他什么。这是完全不可接受的。我们只是把他们踢出去。"马穆拉什维利告诉我，在他的七百名战士中，大约有一百五十人来自东欧以外的地方，并认为美国人的数量为五十人。他说，他的人以小队形式行动，攻击俄罗斯的供应线。在侦察期间，他们徒步行动，避免使用无线电或手机。他们可能会在森林里待上一周，观察道路并追踪俄罗斯车辆，然后再发起伏击。他说，他们刚刚杀死了60名俄罗斯人，并在鲁德尼茨克村缴获了三辆坦克。他给我看了他手机上的行动录像，并抱怨说乌克兰安全官员总是审查他的TikTok视频。
马穆拉什维利说，泽伦斯基关于国际军团的宣布 "搞砸了一切"。"这成了一场狗屎秀，因为他们没有为这么多志愿者做好准备。"由于没有找到任何可观的外国旅，成群结队的过剩英语人才试图在格鲁吉亚的部队中找到一席之地。"马穆拉什维利说："我们没有时间开始训练或组建新的小队。几周后，他暂停了对西方人的招募。"他说："我现在只招收格鲁吉亚人。"对于格鲁吉亚人，我可以解释说，你必须等待两周才能得到武器。我不能对美国人说这些。他们太没有耐心了。"
 
另一支已知接受外国战斗人员的部队是极右翼的阿佐夫营，这是一支精英民兵，其新异教、准法西斯主义美学和雅利安人至上的意识形态长期以来使其成为支持乌克兰的西方自由派的尴尬。我在基辅工业区郊区的一个旧苏维埃大院找到了阿佐夫的基地，在那里我见到了其创始人安德里-比列茨基。
比列茨基身穿橄榄绿毛衣，腰间挂着一把手枪，在一栋废弃的工厂大楼的混凝土大厅里接待了我，那里的椽子上挂着阿佐夫的黄旗，上面有类似纳粹的狼牙棒标志。他否认他的队伍中曾有过美国人。曾有过一次美国训练员，但仅此而已。"有克罗地亚人，"他告诉我，"白俄罗斯人，格鲁吉亚人，如果我记得清楚的话，还有一些英国人，但不是很多。"
至于国际军团，"它确实存在，"比列茨基说，"但它更多是心理战，"旨在支持全球对乌克兰的支持。他说："这没有实际意义，"他指出，预备役民兵已经有了过剩的人力。
当大部分亚速团在马里乌波尔，被俄军包围，水源不足时，比列茨基来到基辅，寻找新兵。我告诉他，有几十名沮丧的外国志愿者，希望加入任何愿意接受他们的部队。阿佐夫对美国人、加拿大人、英国人之类的人开放吗？比列茨基建议西方人以人道主义身份做志愿者，但他没有回答这个问题。
在我离开的时候，我发现有四个穿着邋遢的准军事装束的人在一个木制看守所旁徘徊。上面用正楷喷上了白色力量的英文字样，还有一个笑脸。我假装打电话，拍了一张照片，但并没有试图与这群人交谈。他们在说英语，但在空转的装甲车的隆隆声中很难听清。我唯一清楚听到的词是 "外国军团"。
在叙利亚报道期间，我参观了外国战斗人员的训练基地，采访了他们的指挥官，并在前线访问了他们的团体，他们在解放拉卡的过程中发挥了微小但值得赞赏的作用。在乌克兰，我了解到，我不会找到如此具体的东西。乌克兰领土防卫国际军团的主要意义，似乎是它代表了乌克兰在信息战争中的胜利。事实上，它是如此成功，以至于俄罗斯发现有必要建立自己的国际部队。入侵乌克兰两周后，普京批准了一项提议，从中东地区部署一万六千名外国军队，与顿巴斯地区的分离主义分子并肩作战，莫斯科通过从叙利亚飞来几百名雇佣兵来支持这一不太可能的计划。
更精明的乌克兰人上演了一场更有说服力的表演。第一批抵达的志愿者经过面试、筛选，获得了适当的合同，并获得了住房。一些人被允许佩戴乌克兰国旗和领土防卫部队的徽章。也许有十几个人已经前往前线，隶属于乌克兰部队，并参加了巡逻。但是，乌克兰人忙于更紧急的事情，并被对亚沃洛夫的打击所吓倒，几乎没有履行泽伦斯基的承诺，即武装、训练和装备所有应他邀请出现的乌克兰朋友。
泽伦斯基的行动是一次出色的战时宣传活动。我遇到的那些在华沙、边境、利沃夫和基辅等地寻找国际军团的西方人，只是不幸的结果。很难不为他们感到有点遗憾。他们中很少有人有大学学位或稳定的工作。他们耗尽了自己的银行账户来购买冬季装备、防弹背心、医疗包和飞机票。少数人可能是出于嗜血或白人至上的动机，但我遇到的大多数人表达的观点都是中间派、自由派或普通的保守派。总的来说，他们只是被吸进了这个场面。那些参加过伊拉克和阿富汗战争的退伍军人，这批人现在已经进入或远远进入中年，他们可能会感到有可能重新获得相关性，并感受到处于世界事件中心的旧有刺激。他们中的许多人仍然在乌克兰，在寻找一些事情--任何事情--做。
他们中的几十人所住的驿站和安全屋并不容易找到，但我设法在圣索非亚大教堂附近的一个改建的艺术馆里找到了一个。这是预备役民兵的一个半官方基地，尽管它感觉更像是一个俱乐部会所。带我进来的23岁的乌克兰人维阿切斯拉夫-德罗法（Viacheslav Drofa）介绍说自己是一名特种部队士兵，但后来澄清说他是一名说唱歌手，正在等待军队的文件通过。他穿着自己设计的制服，防弹背心的前面挂着一把刀和作战手套。
在里面，以外星人和数学公式为主题的艺术作品仍然挂在画廊的墙上。地板上放着被褥、衣服和包，桌子上放着冷比萨，厨房水槽边堆放着锅碗瓢盆。七八个人的军籍我不清楚，他们穿着军装、街服、睡衣和淋浴鞋，来来往往，错综复杂。其中有一个叫威尔的美国人和一个叫罗伯特的波兰人。
他们问我的第一个问题是我是否喝酒。一瓶威士忌站在一张粘稠的桌子上，周围是塑料酒杯。我礼貌地拒绝了（当时是下午两点）。他们的下一个问题是我是否吸食大麻。
我们去了一个防空洞，他们称之为 "吸烟室"。那里有一个用酒瓶浸泡在陶瓷罐中制成的水烟筒，还有一袋名为AK-47的菌种。一个光秃秃的灯泡被安放在一面布满涂鸦的砖墙上。4:20，蘑菇，埃及象形文字，大卫之星，骷髅头和交叉骨。通风口被涂成了静脉喷发的眼球的样子。
 
几个乌克兰士兵被雨淋湿了，进来嗑药。我的翻译，一个名叫伊霍尔的法律系学生，拒绝了我的要求，但我屈服了。我不习惯在战区里发呆，我盯着房间里所有上了膛的枪支，并注意到处理这些枪支时的粗心大意，心里越来越不舒服。一名士兵在他的AK步枪的弹夹上贴了一张未来世界的贴纸，并带着一把匕首，刀柄的形状是野猪头。墙上挂着一条政治横幅，上面画着一个梳着辫子的金发女人。伊霍尔的手臂搭在一个装着大疆Air 2无人机的盒子上，我们后来会把它交给赫尔松的士兵。他的头发被剪掉了，穿着海军风衣和牛血色的Doc Martens。
我转向那个美国人威尔。他不肯说他来自美国的哪个地方，也不说他的姓氏。他的体重不可能超过100磅。他二十多岁，身穿小帽、打底衫和货裤，身上有一种乐观的气息。他有语言障碍，他的一只眼睛略微交叉。虽然他没有当过兵，但他说："我比大多数来这里的军人有更多的持枪经验。"
为了寻找国际军团，威尔曾试图加入格鲁吉亚人，后者把他送到红十字会，而红十字会又给了他基辅一家医院的地址，据说这家医院正在接收志愿者。他在那里也被拒绝了。"他说："我花了好长时间才找到合适的关系。他终于来到了基地，在那里他帮助后备部队向被围困的基辅郊区运送食品、瓶装水和其他物资。这不是前线，但仍然是危险的工作。就在几天前，波兰志愿者罗伯特被一发炮弹引爆后扔进了一堵砖墙。他说，一名医生花了一个小时从他的腿上取出弹片。他的手腕和脚踝仍被包扎着。
滚滚的浓烟笼罩着拥挤的地下室。一些人在咳嗽和窒息，将一把手枪传给别人检查。有人打开了电子音乐。一个瘦小、憔悴、留着大胡子的乌克兰人，名叫朱拉，在一张夹在冰箱和墙壁之间的破椅子上坐下来，手指夹着一根烟，膝盖间夹着一把卡拉什尼科夫。他今年33岁，在加入国土防御部队之前，他是一名不可伪造的代币的开发者。朱拉说，乌克兰人在战争的各个层面都在使用加密货币，从接收捐款到购买靴子、背心和手套。但他说，他们最需要的是反无人机技术，"关于俄罗斯数字和电信系统的良好信息"，以及射频接收器，因为俄罗斯人在前线使用不安全的通道。
卓法打断了我们。他说，现在是时候了，要为奥波隆区的一个由国家资助的老人和残疾人之家提供补给。捐赠的食品已经被装入两辆汽车的后备箱。我和德罗法一起乘坐他那辆改装过的奥迪Quattro，四天后他将在与一辆雷诺车的迎面相撞中撞毁。他摇动引擎，耳朵上挂着一个钢锯形状的耳环，嘴里含着一块巧克力饼干。
副驾驶座上是四十六岁的帕维尔-帕尼奇。"他是个罪犯，"德罗法解释说。帕尼茨几年前从监狱里被释放出来，仍然是某种肉体黑手党中的一个有影响力的人物。在这个非常时期，他的社会贡献是充当红十字会和我所了解的奥波隆的居委会或建筑合作社之间的联络人。他是个秃头，身材苗条，脸色苍白，皮包骨头，戴着一个腰包。他的眼镜带有有色镜片，看起来像护目镜。他微笑着做了一个和平手势。他说："这就是生活，"他说。
我们在荒芜的基辅雨后的街道上穿过红灯，那里几乎所有的商店都关闭了。一些街道标志被喷成了黑色，以迷惑入侵者的视线。俄国间谍在灯柱、建筑物的角落和其他纪念碑上涂上了只有在黑光下才能看到的紫外线油漆--乌克兰人已经抓到了一些破坏者。
第聂伯河两岸是光秃秃的泥地，日落时分，河水看起来好像被蓝色和紫色的染料所饱和。一块广告牌上写着政治家们攥着钱袋，抱着肚子。在每个主要的十字路口，都有被称为捷克刺猬的反坦克障碍物包围的战斗阵地。在我们到达目的地之前，我们不得不出示我们的文件五六次。
这所房子幸运地躲过了炮击。一位名叫伊雷娜-祖伊(Irena Zuy)的妇女下来迎接我们，她穿着一件蓬松的外套。她来自顿涅茨克，俄罗斯人入侵时，她一直住在这里，等待对她背部的肿瘤进行医学成像。"她说："这就像一场恶梦。
远处响起了雷鸣般的声音：三声闷响，滚动的爆炸声。"相当接近，"德罗法说。"但这是我们的部队。当它听起来像暴风雨时，这意味着这是我们这边的大炮在向他们逼近。"帕尼茨与老人们手拉手，给他们拥抱，并合影留念。他让祖伊的女儿笑了起来，有一点。在我们离开之前，他分发了他自己出版的监狱回忆录《放弃希望，所有进入这里的人》的副本。
到这时，很明显，俄国人攻占基辅的攻势已经失败。俄国人的后勤问题、准备不足、战术不当，以及乌克兰人顽强的防御，产生了一个意想不到的结果。3月29日，俄国人宣布从首都北部地区撤军，尽管随着乌克兰部队逐一夺回郊区，城市外围的炮声隆隆和砰砰作响。
4月1日，他们重新占领了布查河上的一个小定居点--扎布奇亚。第二天，我和另外两名记者、两名援助人员以及一名年轻的修理工一起访问了该地区。和我们一起的还有两名新入伍的预备役军人。为我做翻译的法律系学生伊霍尔，以及律师兼咖啡店老板尤里。他们之间有一支步枪，没有制服，只有大衣上的蓝色臂章。我们的三辆车装满了一袋袋土豆、一罐罐腌制的蜜饯、面包和一袋袋糖，准备分发给过去三周一直被俄罗斯占领的扎布奇亚的人们。
天气寒冷而潮湿。高速公路上到处都是被撞坏的汽车和沥青块，履带式车辆将护栏压平。一个加油站曾是一场坦克大战的现场。我们经过的街区看起来就像被小行星击中一样。电缆挂在受损的铁塔上。蜂窝网络已经中断。
离开大路，我们在一个墓地停下，发现了一辆被遗弃的越野车，它的车轮被剥去，车顶上绑着树枝做伪装。驾驶员的一侧布满了弹孔。在不远处的地上躺着被泥土打碎的肠子，还有一大块头骨碎片。尤里用脚把它翻了过来，露出了嗡嗡作响的金发。一只狗晃晃悠悠地走到我们面前，它的头卡在一个塑料容器里，但在我们能帮助它之前，它就溜进了森林。
在扎布奇亚，人们从家里出来迎接我们，看起来惊慌失措，衣衫不整。他们向我们展示了大炮和火箭弹对他们的房屋造成的破坏。他们说，入侵者在附近的森林里扎营。他们烧毁了警察的房子，并威胁要向人们开枪。老妇人哭了，而老人则赞扬了乌克兰的武装力量。
在离开郊区向南的双车道公路上，在四分之一英里的范围内，有十多辆坦克被烧焦、压扁、撞烂，被火烧得面目全非，并被氧化，其中一些还在冒烟。大多数坦克的炮塔被炸掉了，这表明是装备有肩扛式反坦克导弹的乌克兰人所为。道路两边的松树林迷雾重重，使这里成为伏击的理想地点。它闻起来像篝火的残骸。
我们在机械残骸周围徘徊，警惕地雷。我们的修理工拿起一个肯定有30磅重的坦克潜望镜，我们轮流通过取景器查看。我发现地面上有一颗发黑的手榴弹。这时我才注意到旁边烧毁的炮塔里挂着的那个人。
他戴着一顶印有517号数字的软垫油罐车头盔。他的一只胳膊挂在坦克外面，被电线缠住，他的条纹夹克的袖子被撕碎了。一根沉重的金属棒，某种挡板，掉在了坦克的舱门上，阻止了他的战友把他的尸体拉出来。他的上半张脸被烧成了黑色。
一支由乌克兰士兵组成的巡逻队在我们身边停了下来，他们驾驶着被泥浆洗过的尼桑皮卡车队，为死者拍照。他们中的一个人争先恐后地爬上大炮，为他起泡、流血的脸拍了一张特写。其他士兵戴着头盔和护膝在脱臼的炮塔周围徘徊，用手机录制视频。在另一辆被烧毁的车辆--一辆被掏空的装甲运兵车中，他们发现了一个无头俄罗斯人躯干的碳化壳。
其中一名乌克兰士兵--从他的步枪来看，是一名狙击手--戴着一顶报童帽和一块漂亮的手表。另一个人留着长发，戴着塞子耳环。这是我最接近乌克兰正规军步兵的一次，虽然我已经厌倦了纠缠每个人关于美国人的事情，但这终于是我向真正的前线战士询问国际军团的机会。
戴着报童帽的士兵似乎不理解我。他的长发战友记得泽伦斯基的公告，但他在军队中见到的外国人只有爱沙尼亚人、拉脱维亚人和立陶宛人。"我听说过，"他用英语说，"但我从未在现实生活中遇到过。"他们跑去追赶离去的车队，我看着他们向北驶向布查，那里刚刚出现了第一批确认的乱葬岗报告。第二天，政府将宣布整个地区获得解放。基辅之战结束了，我对国际军团痕迹的追寻也结束了。我相信，这只是一个神话，而不是现实，外国志愿者在保卫首都方面的作用几乎为零。尽管西方人渴望在乌克兰作战，但显然不需要他们。
